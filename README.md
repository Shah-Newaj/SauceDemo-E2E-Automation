# SauceDemo-E2E-Automation
SauceDemo Website end2end automation using Playwright, Python and Pytest

(tests and pages both must be packages(not directory) or command will face error)

Command to run the project...
    
    pytest -v -s .\tests\test_saucedemo.py --headed
